# スクロール制限 デモアプリ

```md
# バージョン情報

- node: 16.14.2
- npm: 8.5.0
- yarn: 1.22.18
- react: 18.0.0
```

## 始め方

```bash
git clone git@github.com:yosse95ai/react-scroll-demo.git # ssh
# or
git clone https://github.com/yosse95ai/react-scroll-demo.git # http
```

node のバージョンは v16 以降にする。

```bash
cd react-scroll-demo
yarn
yarn start
```
